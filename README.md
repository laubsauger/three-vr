# three-vr

[Live Demo](https://laubsauger.github.io/three-vr/)

WebXR mixed-reality network visualization app built with Three.js and Vite.

## Deployment

GitHub Pages is deployed by the workflow at [`.github/workflows/deploy-pages.yml`](./.github/workflows/deploy-pages.yml).

In repository settings, use `Settings -> Pages -> Build and deployment -> Source -> GitHub Actions`.
